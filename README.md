# Timely

Menu bar timer app for Mac.

## Features
- Toggle between one off or pommodoro-style between presets
- Two customizable presets (Work / Break)
- App is in menu bar, avaliable at a moment's notice
## Requirements
- macOS 15.0+
- Intel / Appple silicon both supported
